Análise de um conjunto de dados de estudantes em alguns teste.

Após essa análise podemos definir melhor o que representa cada coluna. Isso é importante para essa versão em português.

Assim temos:

  • Gender (Sexo): male/famele;
  • Ethnicity (Etnia) : A,B,C,D e E;
  • Parents Education (Nivel de escolaridade dos pais) : some college, associate's degree, high school, some high school, bachelor's degree, master's degree
  • Lunch (Forma de pagamento da escola) : standart, free/reduce
  • Test preparation course (exame preparatório): Completed/None
  • Math/Reading/Writing Score (notas dos testes): 0-100

Com o objetivo de melhor entendimento, algumas traduções melhores de alguns termos:

  • Some College : Fez facudade mas não finalizou
  • Associate's degree: Curso de até dois anos feito por uma faculdade ou instituição especializada.
  • Hig School: Ensino médio
  • Some high school: Fez ensino médio mas não finalizou.
  • Bachelor's degree: Concluiu o curso de bacharelado.
  • Master's degree: Graduação posterior ao bacharelado, equivalente a dois anos. Similar a mestrado.
  • Math: Matemática
  • Writing: Redação
  • Reading: Leitura

Sobre lunch:

  • Standart: Alunos que possuem a escolar pagar com valor integral
  • Free/Reduce: Alunos que possuem desconto no valor do pagamento ou bolça de estudados, podendo ser integral ou não ao valor

Como os dados simulam um sistema americano de notas vamos converter as notas dos alunos para um sistema de grades, sistema de letras para notas. Com isso podemos dividir os alunos em grupos de acordo com sua performace, porem vamos fazer isso somente com a média final dos alunos

Segundo o sistema americano de notas escolares, as numeração resulta em letras:

  • Score >= 90 - A
  • 90 > score >= 80 - B
  • 80 > score >= 70 - C
  • 70 > score >= 60 - D
  • 60 > score - E/F

Seguindo esse critério, para um aluno ser considerado um excelente aluno, ele tem que possuir a média maior que 90 para ter uma nota A. E para um ser considerado um aluno ruim tem que a média de nota inferior a 60, gerando uma nota E/F. These students have failed in the performance of examens.

Source: https://nces.ed.gov/nationsreportcard/hsts/howgpa.aspx

# Criando a função para a converção
def ScoretoGrade(mscore):    
    if (mscore >= 90 ):
        return 'A'
    if (mscore >= 80):
        return 'B'
    if (mscore >= 70):
        return 'C'
    if (mscore >= 60):
        return 'D'
    else: 
        return 'E/F'

# Criando a coluna com as notas novas    
df['grade'] = df.apply(lambda x : ScoretoGrade(x['mean_score']), axis=1)

Conclução¶

De uma maneira geral a maioria dos estudantes tiveram conceitos aceitáveis em todas as matérias, a distibuição das notas também foi bem similar em todas, mesmo assim a matéria de writing foi a que os alunos tiveram o melhor desempenho e matemática foi o que teve o pior. Se avaliarmos por gênero, o desempenho entre homens e mulheres são muitos similares, sendo que as mulhures tiveram um desempenho levemente melhor. Nas matérias as mulheres tiveram um desempenho melhor em writing e reading, já os homens um desempenho melhor em math, mas nada que fosse muito destoante em cada matéria.

Convertendo as médias dos alunos em grades, notas por letra, organizamos os alunos em grupos de desempenho. Sendo os alunos com conceito A os de melhor desempenho e os de conceito E/F os de pior desempenho, segundo o sistema americano de grades. Embora tivessem a maioria dos alunos uma desempenho acima de E/F, o grupo de alunos E/F foi o que teve a maior quantidade de alunos, um resultado que tem a sua importância.

Quando comparamos as características do alunos "A" e alunos "E/F", podemos ver que a etnia não é um fator relevante já que ambos os grupos tiveram as mesmas características, a maioria dos alunos eram da etnia C e a minoria da etnia E. O mesmo se aplica com a execução do exame de preparação dos alunos para as provas, a maioria dos alunos não fez um exame preparatório.

Porém quando comparamos a forma de financeamento dos estudos, "lunch", vemos que os alunos de conceito "A" em sua maioria tinha a sua eduação não possuia algum tipo de financiamento, já os alunos de conceito "E/F" são praticamentes divididos, porém a maioria ainda não possui algum financiamento também. A comparação do nível de escolaridade tiveram os resultados mais diferentes, os pais dos alunos "A" estudaram em sua maioria além do High School. Essa situalção é oposta aos alunos "E/F", onde a maioria dos pais estudou até o High School ou então não concluiram o High School.
